# 格式模板

```
---
# ✅ 笔记元数据（必写，用于搜索、分类和管理）
title: 笔记标题（和下方主标题保持一致）
author: 你的名字
date: {{date:YYYY-MM-DD}} # 自动生成当前日期（VS Code/Obsidian支持）
update: {{date:YYYY-MM-DD}} # 最后更新日期
category: 分类名称 # 如：技术/UE5、学习/英语、生活/理财
tags: [标签1, 标签2, 标签3] # 细粒度标签，如：[Shader, 渲染, 笔记]
status: 进行中 # 可选：未开始/进行中/已完成/待复习
priority: 中 # 可选：高/中/低
link: [] # 关联笔记：[[另一篇笔记标题]]（Obsidian双向链接）
---

# 笔记标题（唯一，和YAML中title一致）

# 🎯 核心速览（30秒看懂这篇笔记）
> 用3-5个要点概括全文最重要的内容，复习时先看这里
- 要点1：最核心的结论/知识点
- 要点2：必须记住的关键步骤
- 要点3：最容易踩的坑
- 要点4：个人最重要的收获

---

# 📚 正文内容（按逻辑分块，不要超过3级标题）
## 一、基础概念
- 定义：什么是XX
- 本质：XX解决了什么问题
- 作用：为什么要学XX

## 二、核心原理/关键知识点
### 2.1 第一个重要知识点
详细解释，用自己的话复述，不要照搬原文

> 重要内容用引用块高亮，或者用 ==高亮文字==（Typora/Obsidian支持）

### 2.2 第二个重要知识点
- 分点列出，清晰明了
- 每个点只讲一个核心内容
- 避免大段文字

## 三、实操步骤/案例
1. 第一步：做什么
2. 第二步：怎么做
3. 第三步：注意什么
```
