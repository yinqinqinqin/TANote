# 自定义管线

## 基础知识

### 常用的材质编辑器默认光照模型所在路径

- ShadingMoudels.ush文件
- DefaultLitBxDF函数中

![image-20260702111354081](/Users/bytedance/Desktop/PracticeSpace/BlogBuild/img/image-20260702111354081.png)

```
FDirectLighting DefaultLitBxDF(
    FGBufferData GBuffer,
    half3 N,
    half3 V,
    half3 L,
    float Falloff,
    half NoL,
    FAreaLight AreaLight,
    FShadowTerms ...
)
```

| 参数           | 含义                                                     |
| -------------- | -------------------------------------------------------- |
| `GBuffer`      | 延迟渲染里的材质信息（BaseColor、Roughness、Metallic等） |
| `N`            | 法线（Normal）                                           |
| `V`            | 视线方向（View Dir）                                     |
| `L`            | 光照方向（Light Dir）                                    |
| `NoL`          | `dot(N, L)`，法线和光方向夹角                            |
| `Falloff`      | 光照衰减                                                 |
| `AreaLight`    | 面光源数据                                               |
| `FShadowTerms` | 阴影信息                                                 |
